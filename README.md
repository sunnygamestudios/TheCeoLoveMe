The CEO Love Me Enable special content
into game scence ,CTRL+ALT+L to open the console and enter "enable r18"
（If the mouse is not visible, press Tab）. 
If it shows that the cheat code is enabled, it means it is successful.

The CEO Love Me 特別コンテンツを有効にする
ゲームシーンに移動し、CTRL+ALT+L を押してコンソールを開き、「enable r18」と入力します。
（マウスが表示されていない場合は、Tab キーを押してください）。
チートコードが有効になっていることが表示されれば成功です。
